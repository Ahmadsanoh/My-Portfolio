Application Android sécurisée d’entraînement cognitif

Braincoach

Application mobile native d’entraînement cérébral intégrant authentification sécurisée, base de données temps réel Firebase et modules interactifs de calcul algorithmique

Interface Utilisateur & Design Android

Expérience Utilisateur

Authentification

Flux d'inscription sécurisé avec validation dynamique des champs (email, téléphone) en temps réel.

Gamification

Accès instantané à trois tests cognitifs : Triangle Math, Low to High, et Quick Calculation.

Synchronisation

Récupération fluide des données utilisateur depuis Firebase Realtime Database après chaque login.

Fonctionnalités de l'App

  • Firebase Auth : Système de gestion des identifiants robuste avec gestion automatique des erreurs et feedback contextuel.
  • Puzzles Mathématiques : Logique métier complexe développée en Kotlin pour les tests de calcul rapide et logique spatiale.
  • Realtime Database : Architecture NoSQL pour le stockage méthodique des profils et des scores de progression.
  • Native UI : Interface réactive utilisant les Intentions Android pour une navigation fluide entre les modules d'entraînement.

Stack Technique

Langage Kotlin / Android SDK
Environnement Android Studio
Backend / Auth Firebase Services
Database Firebase Realtime DB
